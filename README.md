# Android-qCON_Display_Tablet

Proprietary app to visualize and recording of data from the qCON-NOX monitor used to monitor consciousness during anesthesia

For further information of the app and the system you can visit next web page: https://quantiummedical.com/products/conox/
and https://play.google.com/store/apps/details?id=qm.display.qnoxqcon

App features:
- For Android phone or tablet
- Bluetooth connection with the monitor qCON
- USB connection with the monitor qCON
- Text to speech
- Can change the languages, English, Spanish, German
- Save data from monitor qCON

Note: for avoid confict with the company not all the files are shared in the repository
